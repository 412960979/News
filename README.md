News:阅读类软件
============================
News是一款开源阅读类APP。项目采用MVP模式开发，通过动画和自定义的UI来提高阅读体验。一共分为4个模块：图片，文章，音乐，影视。

![](https://github.com/412960979/News/blob/master/%E6%BC%94%E7%A4%BA.gif)

目前1.0版本支持的功能
----------------------------
* 首页4大模块（图片，文章，音乐，影视）
* 支持查看个模块详情。支持播放音乐和影视
* 支持分享，查看点赞数和评论数
* 添加了本地缓存
* 部分图标使用了Vector更好的适配不同屏幕
* 大量的动画效果


项目中用到的第三方类库
----------------------------
* Glide
* RxJava
* retrofit
* retrolambda
* vitamio
* palette
* UI设计部分参考了Plaid


未来2.0版本将会支持的功能
----------------------------
* 阅读、音乐、影视列表
* 自定义主题（高度自定义app ui风格）
* 添加点赞、评论以及评论列表
* 登录功能
* 图片缩放拖动，图片下载
* 添加部分操作的动画
* 短视频的压缩
